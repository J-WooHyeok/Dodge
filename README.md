# 구현된 기능 <br/>
* 각 Bullet과 Player는 collider를 이용하여 충돌 감지
* Bullet을 0.5초에서 3초 사이에 랜덤으로 생성
* input.GetAxis를 이용하여 키보드로 수평축, 수직축의 입력 감지
* Player Die 이후 Canvas를 SetActive(true) 로 전환하여 Restart 버튼과 Quit버튼 구현
